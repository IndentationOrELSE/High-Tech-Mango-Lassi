Fonts

Use this code in the head section of your html:

<link rel="stylesheet" href="https://hightechmangolassi.multicoder2021.repl.co/styles/fonts.css">

Then you can simply select from tens of fonts with this code:

font-family: 'font-family-name'

You can also change the font thickness (100 to 900 for most families):

font-weight: 100, 200, 300, 400, 500, 600, 700, 800 or 900

Additionally, some fonts :

font-weight: 100, 200, 300, 400, 500, 600, 700, 800 or 900

You can see the available families and weights at source code
